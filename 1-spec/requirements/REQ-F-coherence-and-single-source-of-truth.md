# REQ-F-coherence-and-single-source-of-truth

**Status:** Draft
**Priority:** Should-have
**Type:** Functional
**Source:** GOAL-coherent-narrative

## Description

The system shall maintain a persistent, continuously revalidated project narrative that functions as a single source of truth while remaining corrigible when reality changes.

## Acceptance Criteria

- [ ] Given newly ingested knowledge that changes understanding, the system updates its maintained narrative
- [ ] Given a proposed interpretation conflicting with broader context, the system flags or revises it before promoting as truth
- [ ] Given a request for current project state, the system provides the narrative with traceable links to sources and uncertainties
