# GOAL-coherent-narrative

**Description**: Maintain a persistent, continuously revalidated project narrative that functions as a single source of truth — updated when reality changes, flagging interpretations that conflict with broader context.

**Status**: Draft

**Priority**: Should-have

**Source stakeholder**: [STK-ai-agent](../stakeholders.md), [STK-team-onboarder](../stakeholders.md)

## Success Criteria

- [ ] New knowledge that changes project understanding triggers narrative update
- [ ] Coherence checks flag interpretations conflicting with broader context before promoting as truth
- [ ] Project state responses include traceable links to sources and identified uncertainties
