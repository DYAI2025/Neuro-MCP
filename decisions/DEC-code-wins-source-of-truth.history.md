# DEC-code-wins-source-of-truth — History

## 2026-04-10 — Initial Decision

**Context:** NeuroMCP maintains two parallel indices (brain notes and code). When they disagree, a clear priority rule is needed.

**Decision:** Code wins as source of truth.

**Alternatives considered:**
- Brain wins (rejected: notes go stale, code is always current)
- Equal weight (rejected: ambiguity in contradiction resolution)
- User chooses per query (rejected: added complexity, inconsistent behavior)

**Accepted by:** Developer (project owner)
